Quick Start

Up and running in under a minute

Install from npm or load from CDN — choose the method that fits your workflow.

Install via npm
npm install js-cloudimage-smart-crop-preview
import { CISmartCrop } from 'js-cloudimage-smart-crop-preview';
import 'js-cloudimage-smart-crop-preview/css';

new CISmartCrop('#my-viewer', {
  src: 'https://example.com/photo.jpg',
  presets: [
    { name: 'landscape', ratio: '16:9' },
    { name: 'square', ratio: '1:1' },
    { name: 'portrait', ratio: '9:16' },
  ],
});
Use via CDN
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/js-cloudimage-smart-crop-preview/dist/index.css">
<script src="https://cdn.jsdelivr.net/npm/js-cloudimage-smart-crop-preview/dist/index.umd.js"></script>

<div data-ci-smart-crop
  data-ci-smart-crop-src="photo.jpg">
</div>

<script>CISmartCrop.autoInit();</script>
Features

Everything you need for smart cropping

Drag the focal point and see all crop previews update in real time.

🎯 Focal Point

Click, drag, or use keyboard arrows to set the center of interest. All crops follow.

🖼 Multi-Preset Preview

See 16:9, 1:1, 9:16, 4:5, OG Image, and custom ratios — all at once.

🎨 Overlay Rectangles

Hover a preview card to see exactly where the crop falls on the original image.

Integration

Embed in any project

Add smart crop preview to your e-commerce admin, CMS, or any web application in minutes.

E-Commerce / CMS

Product image editor with focal point — crop previews update live. Use onChange to save crop data to your backend, exportCropBlob() to generate thumbnails client-side.

const cropper = new CISmartCrop('#product-image', {
  src: product.imageUrl,
  presets: [
    { name: 'card', ratio: '4:3', label: 'Product Card' },
    { name: 'thumb', ratio: '1:1', label: 'Thumbnail' },
    { name: 'banner', ratio: '21:9', label: 'Banner' },
  ],
  onChange: ({ focalPoint, crops }) => {
    // Save to your API
    fetch('/api/products/' + product.id + '/crops', {
      method: 'POST',
      body: JSON.stringify({ focalPoint, crops }),
    });
  },
});

// Download a specific crop
const blob = await cropper.exportCropBlob('thumb', 'webp', 0.85);
uploadToStorage(blob);
React Integration

First-class React wrapper with props, refs, and hooks. Syncs with your component state automatically.

import { CISmartCropViewer } from
  'js-cloudimage-smart-crop-preview/react';
import 'js-cloudimage-smart-crop-preview/css';

function ProductEditor({ imageUrl }) {
  const viewerRef = useRef(null);

  const handleSave = async () => {
    const crops = viewerRef.current.getCrops();
    const blob = await viewerRef.current
      .exportCropBlob('thumb', 'webp');
    await uploadCrops(crops, blob);
  };

  return (
    <CISmartCropViewer
      ref={viewerRef}
      src={imageUrl}
      presets={[
        { name: 'card', ratio: '4:3' },
        { name: 'thumb', ratio: '1:1' },
      ]}
      onChange={({ crops }) => console.log(crops)}
    />
  );
}
